Introduction

Programmatic access to the V-AUTON foundation model.

The V-AUTON API provides a RESTful interface to interact with our robotics foundation models. You can generate trajectories, validate URDF files, and run kinematic simulations directly via HTTP requests. All requests must be authenticated via Bearer Token.

Generate Trajectory

Generates a sequence of waypoints based on a natural language instruction and robot constraints.

// Request Example curl -X POST https://api.v-auton.com/v1/trajectory \ -H "Authorization: Bearer YOUR_KEY" \ -H "Content-Type: application/json" \ -d '{ "model": "v-auton-70b", "instruction": "Pick up the red gear and place it on the axle", "robot_config": { "type": "ur5e", "max_velocity": 0.5 } }'
// Response (200 OK) { "id": "traj_8923nsd2", "status": "success", "waypoints": [ [0.23, 0.45, 0.12, 0.0, 1.57, 0.0], [0.24, 0.46, 0.15, 0.0, 1.57, 0.0] ], "physics_check": true }
